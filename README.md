## 🚀 Installation locale (Pour les développeurs)
Si vous souhaitez cloner et exécuter ce projet localement sur votre machine :

1. Clonez le dépôt :
\`\`\`bash
git clone https://github.com/ELMOHAMMADY-YOUSSEF/smiyat-repo-dyalk.git
\`\`\`

2. Installez les dépendances :
\`\`\`bash
npm install
\`\`\`

3. Lancez le serveur de développement :
\`\`\`bash
npm run dev
\`\`\`
